# Student Management System (Java OOP)

A console-based Student Management System developed using Java
to practice Object-Oriented Programming concepts.

## Features
- Add student (unique roll number validation)
- Remove student by roll number
- Search student by roll number
- View all students
- Menu-driven console UI

## Concepts Used
- OOP (Encapsulation, Classes, Objects)
- ArrayList
- Loops & Conditionals
- Input validation
- Basic menu-driven program

## How to Run
1. Clone the repository
2. Compile all `.java` files
3. Run `Main.java`
